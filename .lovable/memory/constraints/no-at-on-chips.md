---
name: No @ symbol on chips
description: Chip labels must NOT show @ prefix — just the plain label (e.g. "Clara" not "@Clara")
type: constraint
---
Do not display the @ symbol on reference chips in prompt boxes. The @ mention trigger for inserting chips is fine, but the rendered chip label should be plain (e.g. "Clara", "img1", "Script V1").
**Why:** User explicitly rejected @ prefix on chips.
