# Memory: features/create-page/audio-mode-tools
Updated: now

In audio/music mode, the toolbar has these specialized buttons:
- **Genre**: Uses Music icon, visual grid of 15 genres with thumbnails
- **Style**: Uses Headphones icon, options for "Instrumental" or "Vocals". When Vocals is selected, shows Male/Female voice picker and lyrics textarea with AI Writer option
- **Title**: Uses PenTool icon, popover with text input for track title
- **Voice**: Uses Mic icon, replaces Character button, shows voice selection list (Rachel, Aria, Roger, etc.)
- **Reference**: Uses AudioLines icon, replaces standard Reference with options for Upload Audio, Record Audio, and Remix Audio
- **Model Version**: Titled 'Model Version', offers V4/V4.5/V5 options with 'V4 (Basic)' as default
